SoundWave - Wireless Headphones Landing Page

 Project Overview

The SoundWave Wireless Headphones Landing Page** is a responsive product landing page developed as part of the Week 02 Minor Project for the Web Development course.

This project showcases a modern landing page for a wireless headphones product using HTML5 and CSS3. It demonstrates responsive web design principles, clean layouts, and attractive user interface elements.



 Features

- Responsive Navigation Bar
- Hero Section with product image and description
- Product Features section with feature cards
- Benefits section highlighting product advantages
- Pricing section with launch offer
- Customer Reviews section
- Contact information in the Footer
- Smooth scrolling navigation
- Mobile, Tablet, and Desktop responsive design



 Technologies Used

- HTML5
- CSS
- Flexbox
- CSS Grid
- Responsive Web Design

 Responsive Design

The landing page is fully responsive and optimized for:

- 💻 Desktop
- 📱 Mobile
- 📟 Tablet

The layout automatically adjusts to different screen sizes using Flexbox and CSS Grid.



 How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Open the `index.html` file in any modern web browser.

No additional installation or setup is required.
